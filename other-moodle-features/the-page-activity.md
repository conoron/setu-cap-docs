# The Page Activity

The Moodle page activity is a nice feature, a page resource creates a link to a screen that displays the content created by the teacher. The Text editor allows the page to display many different kinds of content such as plain text, images, audio, video, embedded code, or a combination of all of these. Pages are more accessible than uploading word-processed documents, particularly if the document just contains text to be read and not downloaded.
