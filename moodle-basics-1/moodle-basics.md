# Moodle Basics

### Modules or Courses? <a href="#modules-or-courses" id="modules-or-courses"></a>

In SETU, we use terms like "course" and "programme" interchangeably. Moodle terminology uses 'course' as the term for a teaching space; in SETU, we call that space a Module. For these guides, we will use the term course because we will be using [moodle.org](https://www.moodle.org) content.

We also use non-teaching spaces in Moodle, and we call those spaces 'Information Areas'. All the same functionality is available in Information Areas as standard teaching spaces; the difference is that there is a corresponding CRN in Banner to match the Module in Moodle; however, Information just lives in Moodle.
