# Moodle Basics

## Modules or Courses?

In SETU, we use terminology like course and programmes interchangeably. Moodle terminology uses 'course' as the term for a teaching space; in SETU, we call that space a Module. For the purpose of these guides, we will use the term code because we will be using [moodle.org](https://www.moodle.org) content.

We also use non-teaching spaces in Moodle, and we call those spaces 'Information Areas'. All the same functionality is available in Information Areas as standard teaching spaces; the difference is that there is a corresponding CRN in Banner to match the Module in Moodle; however, Information just lives in Moodle.

### Courses

Moodle courses/modules will have a CRN (5-digit number) built into the title of the course, the full course title will consist of the module name as it exists in the student registration system (Banner), the CRN, and the academic year \[20XX-20XX], so your course will look something like Learning to Learn-12345-\[2026-2027].

{% embed url="https://youtu.be/fyCmkyzAhWw" %}

### Course layout.

The course format that we have in place in our instance of Moodle here ate SETU will be '**Custom**', '**Tiles**' and '**Weekly**' format, but you are free to switch the format to one of the moodle original course formats.
