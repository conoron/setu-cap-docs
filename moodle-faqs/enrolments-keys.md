# Enrolments Keys

An enrolment key is a password required to gain access to a module in Moodle that you have not been automatically enrolled on. Please contact the lecturer of the module in question for an enrolment key. Once you have entered the key will be able to view all of the resources in this module area and participate in any activities created there.
