# Can staff members Import content?

Yes, is the quick answer; however, the staff will need to have editing-lecturer access rights on both the source module and the destination module for this to work.

The destination module space should be empty of content (blank) in order for this to work more effectively.

Staff members can do a full import or a partial import; we have more information on [how to import content from one module to another module](https://youtu.be/nGWPuhWA3dw) in this video.
