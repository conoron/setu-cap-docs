# Using wiki's

The WIki activity allows students to create a collaborative document by building pages together, similar to Wikipedia. Unlike other collaborative editing programs (such as Google docs) the Wiki is a standard Moodle activity, and so no extra permissions or logins are needed. Although Wikis are usually collaborative, it is possible to allow each student to create their own Wiki for individual use.
